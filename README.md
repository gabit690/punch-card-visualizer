# Punch card visualizer

## Description

Frontend project that converts uppercase text into a visual punched card to download.
Each characted entered by the user is translated into a hole pattern displayed as a punch card.

## Tech Stack

- Vite
- React
- Typescript
- shadcn/ui
- Tailwind CSS

## Running Locally

Clone the repository:
```bash
git clone repository-url
```

Move into the project directory:
```bash
cd project-folder
```

Install dependencies:
```bash
pnpm i
```

Start the development server:
```bash
pnpm dev
```

Open your browser at http://localhost:5173
